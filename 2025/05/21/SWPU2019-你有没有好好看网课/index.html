# [SWPU2019] 你有没有好好看网课？

打开有两个文件，需要解压，先考虑是不是伪加密，好像不是

若是伪加密，就是 00 00，01 的意思就是确实是加密了

flag3 给了解压密码提示：

掩码一下

拿到密码：

183792

解压出来就是一个 docx 文件和一个音频文件

给了 5.20，711 的提示，估计是在文件的 5.20-7.11 之间有个提示，可以看一下

在 5.63 秒这里有一个信息

..... ../... ./... ./... ../

# 敲击码

以 / 为划分：

..... ../... ./... ./... ../
  5,2     3,1    3,1    3,2
   W       L      L      M

在 7.33 这里又找到了一个信息：

dXBfdXBfdXA=

C:\Users\86182>php -r "var_dump(base64_decode('dXBfdXBfdXA='));"
Command line code:1:
string(8) "up_up_up"

联合一下就是：

wllmup_up_up

swpuctf{A2e_Y0u_Ok?}
