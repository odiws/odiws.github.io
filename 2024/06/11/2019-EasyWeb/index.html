# 2. 利用.htaccess 文件上传

.htaccess 文件提供了针对目录改变配置的方法，在一个特定的文档目录中放置一个包含一个或多个指令的文件，以作用于此目录及其所有子目录。

所以将一句话进行 base64 编码，然后在.htaccess 中利用 php 伪协议进行解码
